    Variable Air Volume Replicator

    Automate the creation of Variable Air Volume (VAV) units using data from an XML file exported from the EcoStruxure WorkStation program and a CSV file. The XML file contains configuration data for a template VAV, while the CSV file contains specific data for multiple VAV units that need to be created. The program processes both files and generates new VAV configurations in accordance with the details provided in the CSV file.
    Input and Output Creator

The XML file contains configuration data for a template I/O point, while the CSV file contains specific data for multiple I/O points that need to be created.
